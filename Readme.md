# Jirajara backend server

This is the backend server for Jirajara, a web application that helps you with your
personal accounting and business.

### Requirements
  - NodeJS 18.x
  - Pnpm
  - PostgreSQL (we tested with 15.1)
  - Redis 6.x
  - Docker
